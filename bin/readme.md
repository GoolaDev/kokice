# bin folder

This is where you put **[mpv.exe](https://mpv.io/installation/)** and **[QuickPHP](https://www.softpedia.com/get/Internet/Servers/Server-Tools/QuickPHP-Web-Server.shtml#download)**

So this folder should look like:
.\bin:
-mpv.exe
-QuickPHP:
---php5ts.dll
---QuickPHP.exe
---QuickPHP.ini
---...
